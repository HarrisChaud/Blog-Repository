I read an article describing the top 10 uses that HTML has been used for in the real world. There are many interesting facts that I learned while reading this article. Such as the fact that every single the page that we see on the world wide web is actually is written using a version of HTML code. Many of the real world uses of HTML include working with the web. I found it interesting that HTML is used in so many aspects that I never considered such as creating responsive images, game development usage, and client-side storage which is feasible by using local storage and indexed.

I also found it interesting that HTML5 is being used to introduce the latest trends in the website creation business. Things such as polyfill libraries, which are supported by old browsers equally as well. HTML5 is also being used for offline capabilities usage. HTML5 has its application cache mechanism, which defines how the browser manages the offline situation. By using a certain property in HTML5, a developer can check if the application is online or not.
